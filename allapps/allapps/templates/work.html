RENS

Replica Exchange with Non-equilibrium switches is an enhanced sampling method propozed by Jarzynski et. al in 2009. It overcomes a few of the constraints imposed by Replica Exchange molecular dynamics, or REMD. Using the equations of motion provided in the 2009 paper, I was able to derive a propagator for this method, and implement the algorithm in python. However, the python code was too slow to generate simulations for a long period of time, and so I rewrote the code in C++. Along with the help of my advisor, Prof. Marimuthu Krishnan, I was able to publish a paper in Journal of Chemical Physics, in October of 2022.

BareBonesBackprop

Implemented a few python classes that serve as building blocks for a neural network. Some of the implemented blocks include Convolution-2d, the linear layer, Flatten, MaxPool, LeakyReLU and Softmax with Cross Entropy Loss. Using these blocks, formed a simple network and trained it on the CIFAR-10 dataset.
